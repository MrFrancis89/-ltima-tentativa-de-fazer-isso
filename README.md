# StockFlow Pro

Aplicativo web progressivo para gerenciamento de estoque e listas de compras.

## Funcionalidades
- Gestão de estoque com categorização automática
- Lista de compras inteligente
- Alertas de estoque mínimo/máximo
- Calculadora integrada
- Reconhecimento de voz
- Tema claro/escuro
- Swipe para ações rápidas
- Exportação/importação JSON
- Compartilhamento via WhatsApp

## Tecnologias
HTML5, CSS3, JavaScript (ES6+), LocalStorage, Web Speech API.

## Como usar
Acesse [https://mrfrancis89.github.io/stockflow](https://mrfrancis89.github.io/stockflow) ou clone e abra o `index.html`.
